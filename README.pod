=head1 INSTALLATION

Plack::Middleware::RequestId installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Plack::Middleware::RequestId

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install
